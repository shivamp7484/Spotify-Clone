# Description:
This is a simple Spotify Clone created using only HTML and CSS.
I built this project to practice webpage layout, responsive design, and basic UI styling.
It includes a homepage design similar to Spotify with sections like navbar, sidebar, playlists, and a music player UI.

# Features:
.Clean and modern UI
.Responsive layout (HTML + CSS only)
.Navigation bar & sidebar
.Playlist cards
.Footer music player design
